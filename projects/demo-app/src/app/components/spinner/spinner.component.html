Spinner

Spinner (Line Type)

The rp-spinner component displays a loading indicator to signal that a process is in progress.

Line Type

When the type property is set to line, the spinner appears as a linear loading bar. This style is useful when you want to show continuous progress in a minimal way, such as at the top of a page or inside a container.

Properties

  • type: Defines the spinner style. Use line for the linear loading bar.
  • size: Not applicable for line type.
  • color: Sets the spinner color. Available values: default, primary, success, warning, danger.

Example

 <rp-spinner 
  class="px-3 block" 
  type="line" 
  color="primary" 
  ></rp-spinner>

Spinner (Circle Type)

The rp-spinner component displays a loading indicator to signal that a process is in progress.

Circle Type

When the type property is set to circle, the spinner appears as a circular loader. This style is useful for indicating indefinite progress, often used in buttons, cards, or full-page loading states.

Properties

  • type: Defines the spinner style. Use circle for the circular loader.
  • size: Controls the diameter of the circle spinner. Accepts a number (in pixels).
  • color: Sets the spinner color. Available values: default, primary, success, warning, danger.

Example

<rp-spinner
  class="px-3 block"
  type="circle"
  [size]="32"
  color="success"
></rp-spinner>
